## Scénario

Vous travaillez comme développeur front-end pour l’agence ArchiWebos qui comprend 50 salariés. 
Ayant terminé votre dernier projet avec un peu d'avance,
vous êtes envoyé en renfort comme développeur front-end d’une équipe qui travaille sur la conception du site portfolio d’une architecte d’intérieur.

## Compétences évaluées

Créez une page web dynamique avec JavaScript

Récupérer les données utilisateurs dans le JavaScript via des formulaires

Manipuler les éléments du DOM avec JavaScript

Gérer les événements utilisateurs avec JavaScript

Récupérer les données d'une API externe

## Installation du backend

Cloner le repo en local
